## 🔧 セットアップ方法

### Electronアプリ (即時利用可能)
```bash
# 現在のWebアプリをネイティブアプリとして実行
npm install
npm run electron:dev
```

### React Nativeアプリ (新規開発)
```bash
# Windows
mobile/setup.bat

# macOS/Linux
cd mobile && chmod +x setup.sh && ./setup.sh
```
