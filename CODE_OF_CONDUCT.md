# Code of Conduct

## 約束

オープンで歓迎的なコミュニティを維持する。年齢・体型・障害・民族・性自認・経験レベル・国籍・人種・宗教・性的指向に関わらず、すべての参加者にハラスメントのない環境を提供する。

## 望ましい行動

- 歓迎的で包摂的な言葉遣い
- 異なる視点・経験の尊重
- 建設的な批評の受容
- コミュニティにとって最善の選択
- 他のメンバーへの共感

## 容認されない行動

- 性的な言葉・画像、性的な注目・誘いかけ
- 嫌がらせ、侮辱、軽蔑的な発言、個人攻撃・政治的攻撃
- 公的・私的な嫌がらせ
- 明示的許可なく他者の個人情報を公開
- プロフェッショナルでない行動

## 強制

メンテナーは適切な行動基準を明確化する責任があり、容認されない行動に対して公正かつ適切な是正措置を取る。

メンテナーは、本規約に従わないコメント、コミット、コード、Wiki 編集、Issue、その他の貢献を削除・編集・拒否する権利と責任を持つ。

## 適用範囲

プロジェクトスペース内、および個人がプロジェクトやコミュニティを代表する公的スペースに適用。

## 報告

虐待的・嫌がらせ・容認されない行動は GitHub Issue (private) または Security Advisory で報告。すべての苦情はレビュー・調査される。

## 帰属

[Contributor Covenant](https://www.contributor-covenant.org), version 2.1 をベースに作成。
