# Artone 動画エディター - クイックスタートガイド

**個人使用向け・5分でセットアップ完了**

---

## 🚀 今すぐ始める

### ステップ1: 必要なソフトウェア

以下をインストールしてください：

1. **Node.js** (バージョン18以降)
   - ダウンロード: https://nodejs.org/ja/
   - インストーラーの指示に従ってインストール
   - 確認: ターミナルで `node --version` を実行

2. **Git** (任意 - ダウンロードする場合に必要)
   - ダウンロード: https://git-scm.com/

### ステップ2: Artoneをダウンロード

```bash
# リポジトリをクローン（Gitを使う場合）
git clone <your-repository-url>
cd Artone

# または、ZIPでダウンロードして解凍
```

### ステップ3: セットアップ

```bash
# 1. 依存パッケージをインストール（初回のみ）
npm install

# 2. 個人使用向け設定ファイルをコピー
cp .env.personal .env.local

# 3. 開発サーバーを起動
npm run dev
```

### ステップ4: ブラウザで開く

ブラウザで以下のURLにアクセス：
```
http://localhost:3000
```

**🎉 完了！動画編集を始められます**

## 📹 初めての動画編集

### 1. プロジェクト作成

1. 「新規プロジェクト」をクリック
2. プロジェクト名を入力（例: 旅行の思い出）
3. 解像度を選択（推奨: 1920x1080）
4. 「作成」をクリック

### 2. 動画ファイルを読み込み

**方法1: ドラッグ＆ドロップ**
- メディアライブラリに動画ファイルをドラッグ

**方法2: ファイル選択**
- 「インポート」ボタンをクリック
- ファイルを選択

**対応フォーマット:**
- 動画: MP4, WebM, MOV, AVI
- 音声: MP3, WAV, AAC

### 3. タイムラインで編集

1. **クリップを配置**
   - メディアライブラリからタイムラインにドラッグ

2. **クリップをカット**
   - 再生ヘッドを切りたい位置に移動
   - キーボードの `S` を押す
   - 不要な部分を選択して `Delete` キーで削除

3. **クリップの長さを調整**
   - クリップの端をドラッグして調整

4. **トランジションを追加**
   - クリップの境目を右クリック
   - 「トランジション追加」を選択

### 4. エフェクトを追加

1. クリップを選択
2. 右側のプロパティパネルで設定：
   - **明るさ**: スライダーで調整
   - **コントラスト**: 色の強弱
   - **彩度**: 色の鮮やかさ
   - **フィルター**: ワンクリックで雰囲気変更

### 5. 音声を調整

1. 音声トラックのクリップを選択
2. **音量調整**:
   - プロパティパネルでボリュームスライダーを調整
3. **フェード効果**:
   - フェードイン/アウトのチェックボックスをON

### 6. エクスポート

1. `Ctrl + E` (Mac: `Cmd + E`) を押す
2. 出力設定を選択：
   - **フォーマット**: MP4（推奨）
   - **品質**: 高（1080p）、中（720p）、低（480p）
3. 「エクスポート開始」をクリック
4. 完了したらダウンロード

## 💡 個人使用向けのヒント

### パフォーマンスを向上させる

1. **GPUアクセラレーションを有効化**
   - Chrome: `chrome://settings/system`
   - Firefox: `about:config` → `layers.acceleration.force-enabled = true`

2. **4K動画を扱う場合**
   - プロキシファイル（低解像度版）を生成
   - 編集中は低画質プレビューを使用
   - エクスポート時に高画質に戻る

3. **メモリを節約**
   - 他のアプリを閉じる
   - ブラウザのタブを減らす
   - 使っていないエフェクトを無効化

### データを安全に保つ

1. **自動保存**
   - デフォルトで30秒ごとに自動保存
   - ブラウザを閉じても作業は保存される

2. **プロジェクトをバックアップ**
   - プロジェクト一覧から「エクスポート」
   - JSONファイルをUSBメモリ等に保存

3. **オフライン作業**
   - 初回起動後はインターネット不要
   - すべてブラウザ内で処理

### よくある質問

**Q: 動画が再生されない**
- ブラウザのGPU設定を確認
- MP4形式に変換してみる
- 別のブラウザ（ChromeまたはFirefox）を試す

**Q: エクスポートに時間がかかる**
- これは正常です（5分の動画で2〜5分程度）
- 品質を「中」に下げると速くなります

**Q: メモリ不足エラー**
- 他のアプリを閉じる
- 動画を短く分割して編集
- ブラウザを再起動

**Q: プロジェクトが消えた**
- ブラウザのキャッシュをクリアしないでください
- プロジェクト一覧の「復元」機能を確認

## 🔒 プライバシーとセキュリティ

### 完全にプライベート

✅ **すべてローカル処理**
- 動画はブラウザ内でのみ処理
- インターネットへのアップロード一切なし

✅ **データ保護**
- プロジェクトはブラウザのローカルストレージに保存
- 他人がアクセス不可能

✅ **トラッキングなし**
- 利用状況の追跡なし
- 個人情報の収集なし

### セキュリティのベストプラクティス

1. ブラウザを最新版に保つ
2. 怪しいブラウザ拡張機能を入れない
3. 定期的にプロジェクトをエクスポート（バックアップ）
