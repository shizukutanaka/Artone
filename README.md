# Artone - Professional Video Editor

Artoneは、ブラウザ上で動作するプロフェッショナルなマルチトラック動画編集アプリケーションです。

## 主な特徴
- ブラウザベースでインストール不要
- マルチトラック編集、リアルタイムプレビュー
- 豊富なエフェクト（ブラー、不透明度、スケール調整など）
- WebM形式による高品質エクスポート
- 直感的なUI

## 使い方
1. リポジトリをクローン
```bash
git clone https://github.com/yourusername/artone-video-editor.git
cd artone-video-editor
```
2. HTTPサーバーを起動
```bash
python -m http.server 8000
# または
npx serve .
```
3. ブラウザで `http://localhost:8000` を開く
